# vue-fake-chat

This project simulates the behavior of chat between browser tabs

## Project stack

- vite
- vue 3
- vue-router
- JavaScript

## Project commands

- node version - 20.11.1
- npm version - 10.2.4

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
